# Pixel briefs - biologi

Inga pixelbilder genereras i detta repo-steg. Denna yta är endast för framtida briefs när OCR, SourceClaims, KnowledgePoints och pixel-bindings är stabila.

Krav före bildarbete:

- accepterad source-lineage per KnowledgePoint
- stabilt pixel-bindings-kontrakt
- samma ProgressionRenderPlan som kemi
- koordinatsystem 650x480 och reference-art-kontrakt
- fem separata granskare med screenshotjämförelse mot kemi-referens
